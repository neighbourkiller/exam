# 在线考试系统数据库表设计

整理依据：`src/main/resources/db/migration` 下的 Flyway 数据库迁移文件，并结合 `src/main/java/com/ekusys/exam/repository/entity` 实体类核对字段。当前迁移脚本主要通过主键、唯一键和普通索引维护数据关系，未声明数据库级 `FOREIGN KEY` 约束，因此本文中的外键关系均按业务代码和字段语义整理为“逻辑外键”。

## 公共字段说明

除个别说明外，各业务表均继承或包含以下审计字段：

| 字段 | 字段含义 |
| --- | --- |
| `create_time` | 创建时间 |
| `update_time` | 更新时间 |
| `create_by` | 创建人用户 ID，逻辑关联 `sys_user.id` |
| `update_by` | 更新人用户 ID，逻辑关联 `sys_user.id` |

## 1. `sys_user`

- 对应业务模块：用户认证与系统用户管理模块。
- 主外键关系：`id` 为主键；`username` 唯一；被 `sys_user_role.user_id`、`student_profile.user_id`、`teacher_profile.user_id`、`teaching_class.teacher_id`、`question.creator_id`、`question_asset.uploader_id`、`paper.teacher_id`、`exam.publisher_id`、`student_teaching_class.student_id`、`exam_session.student_id`、`submission.student_id`、`subjective_grade.teacher_id`、`anti_cheat_event.student_id`、`proctoring_disposition.student_id`、`proctoring_disposition.handled_by`、`operation_audit_log.operator_id` 等字段逻辑引用。

| 字段 | 字段含义 |
| --- | --- |
| `id` | 用户主键 ID |
| `username` | 登录账号，唯一 |
| `password` | 加密后的登录密码 |
| `real_name` | 用户真实姓名 |
| `enabled` | 账号是否启用 |
| `create_time` / `update_time` / `create_by` / `update_by` | 公共审计字段 |

## 2. `sys_role`

- 对应业务模块：权限角色管理模块。
- 主外键关系：`id` 为主键；`code` 唯一；被 `sys_user_role.role_id` 逻辑引用。

| 字段 | 字段含义 |
| --- | --- |
| `id` | 角色主键 ID |
| `code` | 角色编码，如 `ADMIN`、`TEACHER`、`STUDENT` |

| `create_time` / `update_time` / `create_by` / `update_by` | 公共审计字段 |

| 字段 | 字段含义 |
| --- | --- |
